目的:
- UUID を自動で生成し、その内容のQRコードを、Googleドライブにpdfとして保存するwebアプリの実装
- Done定義: 以下の機能が全て動作すること
  - 1ページのみのweb UI
  - UUID を自動で生成もしくは、ユーザーが手動でuuid を入力かを選択できる機能
  - uuidは最大10件まで、自動で出力もしくは、手動で入力できる機能
  - 上のuuid のインプットをqrコードとして、googleドライブapi を利用し、googleドライブの特定のフォルダにpdfで保存する機能
  - google の認証は初回画面描画時に行う
  - 以降は適切な期限のトークンをcookie でもち回して、認証を省略する機能

前提と制約:
- 技術スタック: Nuxt.js。現在のフォルダに作ってください。
- googleドライブapiの利用は、nuxt server dir を利用してください。
- 初回画面描画時の、google認証も nuxt server dir を利用してください。
